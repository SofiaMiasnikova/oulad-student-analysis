# oulad-student-analysis
# Анализ отсева студентов — Open University Learning Analytics

**Кто бросает учёбу и почему?** Исследовательский анализ поведения и удержания студентов онлайн-университета.

---
## О проекте

Проект выполнен на основе датасета **Open University Learning Analytics Dataset (OULAD)** и направлен на выявление факторов, предсказывающих отсев студентов. Анализ проводится с позиции аналитика данных в EdTech-компании, задача которого — улучшить удержание студентов.

**Ключевые вопросы:**

- Какой демографический портрет у студентов, бросающих курс?
- Есть ли критический период отсева в течение курса?
- Предсказывает ли ранняя активность (клики в первые 30 дней) финальный результат?
- Одинаков ли retention rate по разным модулям и семестрам?

---
## Данные

**Источник:** [Open University Learning Analytics Dataset — Kaggle](https://www.kaggle.com/datasets/rocki37/open-university-learning-analytics-dataset)

Датасет содержит анонимизированные данные о 32 593 студентах в 7 таблицах:

| Файл                      | Описание                                            |
| ------------------------- | --------------------------------------------------- |
| `studentInfo.csv`         | Демография и финальные результаты                   |
| `studentVle.csv`          | Ежедневная активность (клики) на учебных материалах |
| `studentAssessment.csv`   | Оценки за задания                                   |
| `studentRegistration.csv` | Даты регистрации и отписки                          |
| `courses.csv`             | Метаданные курсов                                   |
| `assessments.csv`         | Метаданные заданий                                  |
| `vle.csv`                 | Материалы виртуальной среды обучения                |

---

## Структура анализа

1. **Загрузка и типизация** — загрузка всех таблиц, обработка пропусков, оптимизация памяти
2. **EDA — портрет студента** — распределение по полу, возрасту, региону и финальному результату
3. **Воронка и Retention** — когортная матрица удержания по модулям и семестрам, анализ момента отсева
4. **Feature Engineering и статистика** — агрегация кликов по неделям, тест Манна-Уитни для сравнения активных и отсевших студентов
5. **Итоговый дашборд** — GridSpec-дашборд с ключевыми выводами

---
## Ключевые выводы

- Большинство студентов отписываются в **первые 30 дней** курса
- Ранняя активность (клики в 1–2 неделю) — **статистически значимый предиктор** сдачи (p < 0.05)
- Retention rate сильно варьируется по модулям — одни курсы удерживают 70%+ студентов, другие менее 40%
- Студенты **55+ лет** и из районов с **низким социальным индексом** чаще бросают учёбу

4. Запустить все ячейки
